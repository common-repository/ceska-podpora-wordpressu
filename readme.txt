Plugin České podpory WordPressu. Informace, novinky, rady, tipy a triky, vše přehledně na nástěnce v administraci vašeho webu.

== Description ==

Nejnovější příspěvky z fóra České podpory WordPressu a články z webu Separatista.net můžete mít nyní přehledně zobrazeny na nástěnce ve vaší administraci.
